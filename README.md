Run Claude Code, Cursor, Aider, Copilot CLI — all from one cockpit.

CLIDE is a multi-session AI terminal workspace that gives you Mission Control for dozens of concurrent CLI AI agents. Track status, manage sessions, orchestrate agents.

> macOS and Linux support planned.

## Features

- **Mission Control** — Grid dashboard showing all your AI sessions at a glance
- **Fleet Panel** — Tab tree with grouping, emoji tags, and session management
- **Concierge AI** — Auto-respond to terminal prompts with intelligent context
- **Steward Mode** — Autonomous agent orchestration across multiple AI agents
- **Intel Panel** — Real-time insights aggregated across all sessions
- **Multi-window** — Tear out panels, run across monitors
- **Auto-update** — Seamless background updates, never miss a release
